# オーナー送金・月次締めマネージャー（owner-payout-tracker）

賃貸・ビル管理の**月次経理サイクル**を「集金日別」に進捗管理し、**オーナー報告書**を自動生成するツール。
これまで完了ファイルに「☑」を付けて手管理していた締め状況を、本物の進捗ボードに置き換える。

- port: **8519**
- スタック: Streamlit + SQLite（ブラウザ完結・ローカル保存）

## 月次5ステップ

`① 請求書発行 → ② 郵送・投函 → ③ 入金確認 → ④ オーナー報告書 → ⑤ 送金・送付`

各管理先ごとにチェックして進捗を可視化。締め日（集金日+3日）を過ぎて未完のものは🔴アラート表示。

## 機能

1. **📋 月次ダッシュボード** — 集金日（10日/15日/16日/20日/25日…）別に管理先を並べ、5ステップを☑管理。全体進捗・締切超過アラート付き。
2. **📊 オーナー報告書 自動作成** — 集金額・管理手数料率・経費（修繕/立替/広告料）を入力 → お振込額を自動計算 → 報告書Excelをダウンロード。保存すると④が自動で☑に。
3. **🏘 物件マスター** — 37管理先の一覧（集金日・オーナー・郵送方法）。

## データ

`seed_properties.json` … `管理先・入居者一覧_2026-07.xlsx` の
「②毎月請求(支払日別)」＋「④郵送投函方法」から生成した実データ（37管理先）。
初回起動時に `tracker.db`（SQLite）へ投入。進捗・報告書は同DBに保存。

## 起動

```bash
cd owner-payout-tracker
./run.sh          # 初回は venv 作成＋依存インストール
# → http://localhost:8519
```
