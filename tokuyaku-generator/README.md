# 📑 特約条項ジェネレーター

不動産売買契約の**特約条項**を、目次から選んでAIで本文を生成し、並べ替えて Word / テキストで書き出す業務支援アプリ。

元資料「特約文目次（資料9-1）」の **31カテゴリ・170項目** を内蔵しています。

## 使い方の流れ

1. **選ぶ** — 左の目次（カテゴリ／キーワード検索）から特約項目を「＋」で追加
2. **生成** — 各項目の「本文を生成」または「未生成を一括生成」でAIが条文の下書きを作成
3. **整える** — 並べ替え（↑↓）・編集・「追加の事情」を入力して再生成
4. **書き出す** — Word（.docx）／テキスト（.txt）でダウンロード

サイドバーで物件・売主・買主の表記、文体（である調／ですます調）を指定すると本文に反映されます。

## 起動

```bash
cd tokuyaku-generator
pip install -r requirements.txt
streamlit run app.py --server.port 8513
```

- AI生成はローカルの `claude` CLI（`~/.local/bin/claude`、`--model sonnet`）を呼び出します。**APIキー不要**。
- `claude` が見つからない場合は本文を手入力すれば出力まで利用できます。

## ⚠️ リーガルチェック必須

AIは誤ることがあり、直近の法改正に未対応の場合があります。出力は**下書き**です。
必ず専門家が確認し、契約書本文と表記（送り仮名・文体）を統一してください。

## ファイル

| ファイル | 役割 |
