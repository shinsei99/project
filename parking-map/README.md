# 駐車場 配置図ビューア（parking-map）

駐車場の**配置図（区画割り・車室の向き・傾き）**をSVGで再現し、区画クリックで契約者/賃料/保証金/契約日を表示するビューア。空き=緑 / オーナー使用=黄 / 契約中=白 / 青空=青枠 で色分け。

## 設計方針（重要）

- **車室レイアウトは固定**（`template.html` にハードコード：区画位置・向き・傾き・青空配置・入口）
- **空き状況・契約者などの中身は起動の度にレントロールxlsxから読み込む**（空きは変動する前提）

## 構成

| ファイル | 役割 |
|---|---|
| `template.html` | 固定レイアウト＋占有データ差し込み枠 `__DATA_JSON__` |
| `serve.py` | 起動の度にxlsxを読み、`区画No→占有状況` を差し込んで配信（**port 8522**） |
| `起動.command` | ダブルクリックで起動＋ブラウザ自動オープン |

※ `index.html`（個人名ハードコードの静的版）と `index_backup_*.html` は `.gitignore` 済み（公開しない）。

## 起動

依存：`openpyxl`（未導入なら `pip3 install -r requirements.txt`）

```bash
python3 serve.py         # → http://localhost:8522 が自動で開く
# または 起動.command をダブルクリック
```

レントロールを編集して**保存**→ブラウザを**再読込（⌘R）**すれば色・詳細が最新に反映される（再起動不要）。

## データ元（レントロール）

```
…/共有フォルダ/（★必読★）新共有フォルダ/物件・管理/レントロール一覧（駐車場他）.xlsx
```

シート `角屋（横堤）モータープール` を読む。判定：現況=空室→空き / 契約者に「オーナー」→オーナー使用 / それ以外→契約中。
※パスは `serve.py` 冒頭の `XLSX` 定数。メインPCでパスが違う場合はここを修正。

## 第一号：角屋(横堤)モータープール

- 北側BOX 19台（1〜21・欠番4,9）＋ 中央 青空5台（27,26,25,23,22／**24は不在**）＋ 南側BOX 17台（30〜47・欠番42）＝ 全41区画
- 入口は区画30の左スペース
- **正の作図元**：`…/塚本/P配置図面(本庄西・大京 ・横堤）.xls` の「横堤」シート（青空は下段区画の真上に積む形）。
  ※ `角屋横堤モータープール.xlsx`（スキャン案内書）とは別物。**.xlsが正**。
