# 駐車場配置図ビューア（parking-map）

駐車場の**配置図（区画割り・車室の向き・傾き）**をSVGで再現し、区画クリックで契約者/賃料/保証金/契約日を表示するビューア。空き=緑 / オーナー使用=黄 / 契約中=白 / 青空=青枠 / 時間貸=グレー で色分け。ページ最上部の切り替えバーで複数の駐車場を切り替えられる（現在7件）。

## 設計方針（重要）

- **車室レイアウトは固定**（`template.html` にハードコード：区画位置・向き・傾き・青空配置・入口）
- **空き状況・契約者などの中身は起動の度にレントロールxlsxから読み込む**（空きは変動する前提）
- 駐車場ごとに追加設定は `template.html` の `LOTS_CFG`（表示用）と `serve.py` の `LOTS`（データ取得元）の両方に追記する

## 構成

| ファイル | 役割 |
|---|---|
| `template.html` | 固定レイアウト（駐車場ごとの描画関数＋`LOTS_CFG`）＋占有データ差し込み枠 `__ALL_DATA_JSON__` |
| `serve.py` | 起動の度に各駐車場のxlsxを読み、`区画No→占有状況` を差し込んで配信（**port 8522**） |
| `起動.command` | ダブルクリックで起動＋ブラウザ自動オープン |

※ `index.html`（個人名ハードコードの静的版）と `index_backup_*.html` は `.gitignore` 済み（公開しない）。

## 起動

依存：`openpyxl`（未導入なら `pip3 install -r requirements.txt`）

```bash
python3 serve.py         # → http://localhost:8522 が自動で開く
# または 起動.command をダブルクリック
# launchd常時起動時は python3 serve.py --daemon （ブラウザ自動オープンを抑制）
```

レントロールを編集して**保存**→ブラウザを**再読込（⌘R）**すれば色・詳細が最新に反映される（再起動不要）。
URLの `#lot-id`（例: `#daikyo`）で特定の駐車場を直接開ける。

## データ元（レントロール）

`serve.py` の `LOTS` 辞書に、駐車場ごとに `file`（xlsxパス）・`sheet`（シート名）・`start`/`end`（データ行範囲）・`cols`（No/現況/契約者/区分/賃料/保証金/契約日の列番号）を設定。判定：現況=空室→空き / 契約者に「オーナー」→オーナー使用 / それ以外→契約中。

- 角屋(横堤)・大京・本庄西・十三・鴫野東2丁目第2・永和3丁目 → `…/物件・管理/レントロール一覧（駐車場他）.xlsx`
- コーポ・ラ・ベリエール → `…/物件・管理/レントロール一覧（マンション）.xlsx` の「駐車場」セクション（号室シートとは別列レイアウト）

※メインPCでパスが違う場合は `serve.py` 冒頭の `DROPBOX_ROOT` を修正。

## 第二号：大京モータープール

- 全22区画：番号区画 10-16（上段）・1-6（下段）、BOXノーマル N1-N5（縦横やや短め）、BOXスリム S1-S4（ノーマルよりさらに横短め）
- 上段（10-16・N3-N5）は直線配置。下段（1-6・S4-S1・N2-N1）は「1番からN1まで一本の斜め帯」として連続配置し、左から右上がりに傾く（元図面の`大京 (最新）`シート準拠）
- 図形は常に直角の四角形（せん断による菱形化はしない）。行全体を進行方向ベクトル＋直交ベクトルでリジッドに傾けている（`diagRow`関数）
- 7・17・18番は廃番済み（レントロールから削除済み）

## 第三号：コーポ・ラ・ベリエール駐車場

- 全7区画（1〜7番）、均等サイズで一列に並ぶシンプルな配置

## 第六号：鴫野東2丁目第2駐車場

- 5列に簡略化（実際の敷地は斜めだが論理配置は直線で表現）：`{25,24,23,22}縦列`（間隔広め）→`{26}単独・縦置き`→`{1,2,3}上段/{4,5,6}下段の2段グリッド`→`{7〜14}縦列8台`→`{15〜21}縦列7台`＝全26区画

- 正の配置図：`…/管理物件/駐車場/A-3476 鴫野東2丁目第2駐車場/_配置図.pdf`（敷地全体が斜めに描かれているため、PyMuPDF (`fitz`) で高解像度レンダリング＋クロップして読み取った）

## 第七号：永和3丁目駐車場

- 全7区画。元図面は縦長レイアウトだが見やすさのため時計回りに90度回転して配置：左から `{1,2}横置き・縦積み` → `{3,4,5}縦置き・横並び` → `{6,7}横置き・縦積み・下にズレる`

- 正の配置図：`…/管理物件/駐車場/E-3482 永和3丁目駐車場/_配置図.pdf`
