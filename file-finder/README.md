# 横断ファイル検索ブラウザ (file-finder)

社内共有ドライブの棚卸しExcel（`全ファイル一覧.xlsx`）を読み込み、**11,000件超のファイルをフォルダ階層を辿らず横断で即検索**するツール。

## できること
- ファイル名・フォルダパス・推定物件名を対象に **スペース区切りAND検索**
- **カテゴリ / 拡張子 / 更新日範囲** で絞り込み、更新日・サイズ・名前で並び替え
- 行クリックで詳細パネル → **フルパスをコピー**して共有ドライブで開く
- ヒット件数・合計容量を表示、**検索結果をCSV書き出し**
- サイドバーから最新の棚卸しExcelに差し替え可能

## データ
- `data/全ファイル一覧.xlsx`（5カテゴリ：営業・募集／管理業務／業者・設備管理／経理・請求書／社内・総務）
- 列：種別・フルパス・名前・親フォルダ・階層・拡張子・サイズ・サイズ(bytes)・更新日時
- ※ ファイルの「中身」ではなくメタデータのみ
- ⚠️ **このExcelは物件名・取引先名を含むため、公開リポジトリにはコミットしない**（`.gitignore`で `data/` を除外）。
- 実際のフルパスは各カテゴリ名（トップ階層フォルダ）を先頭に付けて表示・コピーされる（例：`営業・募集/物件資料/…`）。

## 別PCでの引き継ぎ（アップロード方式）
1. `git pull` でコードを取得（`data/` のExcelはgitに含まれない）
2. `python3 -m streamlit run app.py --server.port 8520` で起動
3. データが無いので画面上部のアップローダーが自動で開く → **全ファイル一覧.xlsx** を選択すれば検索開始
   - 常設したい場合は Excを `file-finder/data/全ファイル一覧.xlsx` に置けば、次回から自動読み込み（アップロード不要）

## 起動
```bash
cd file-finder
pip install -r requirements.txt
python3 -m streamlit run app.py --server.port 8520
```
→ http://localhost:8520
